Introduction

Welcome to the Crateria ecosystem documentation. Crateria is a suite of modern, security-first Linux desktop tools written entirely in Rust.

Our focus is on high performance, memory safety, and modern desktop integrations (such as Wayland and the COSMIC Desktop environment).

The Suite

Product Type Description
Trance Daemon / UI A Wayland-native modular screensaver system with COSMIC support.
Morphball CLI / TUI A secure, multi-threaded archive manager with Landlock sandboxing.

Installation

Crateria applications are distributed through signed native package repositories for APT and DNF package managers.

Fedora (DNF)

Shell Command
sudo curl -fsSL https://crateria.github.io/packages/rpm/crateria.repo -o /etc/yum.repos.d/crateria.repo
sudo dnf install trance morphball

Debian / Ubuntu / Pop!_OS (APT)

Shell Command
sudo mkdir -p /etc/apt/keyrings
sudo curl -fsSL https://crateria.github.io/packages/apt/crateria-keyring.gpg -o /etc/apt/keyrings/crateria.gpg
echo "deb [arch=amd64 signed-by=/etc/apt/keyrings/crateria.gpg] https://crateria.github.io/packages/apt stable main" | sudo tee /etc/apt/sources.list.d/crateria.list
sudo apt update && sudo apt install trance morphball

Trance Configuration

Trance is configured via a TOML file located at ~/.config/trance/config.toml.

Configuration Options

~/.config/trance/config.toml
idle_enabled = true
idle_timeout_mins = 5
active_saver = "beams"
show_fps_overlay = false
render_scale = 1.0

D-Bus API Reference

The screensaver daemon registers the io.github.ubermetroid.trance service on the D-Bus Session Bus.

Interfaces

Interface Method Type Description
io.github.ubermetroid.trance get_status() Method Returns a map of the current daemon states.
io.github.ubermetroid.trance enable() Method Privileged Enables screensaver monitoring. Requires control-peer authorization.
org.freedesktop.ScreenSaver SetActive(bool) Method Privileged Forces screensaver state. Requires control-peer authorization.

Plugin Development

Trance screensavers are modular plugins compiled as dynamic library assets. Developers can build custom effects using the trance-api library interface.

Basic Boilerplate

src/lib.rs
use trance_api::{Plugin, PluginContext, RegisterPlugin};

struct CustomEffect;

impl Plugin for CustomEffect {
    fn render(&mut self, ctx: &mut PluginContext) {
        // Your custom Wayland rendering code here
    }
}

Morphball Usage

Morphball provides a secure command-line tool and an interactive terminal user interface (TUI) for archive compression and extraction.

CLI Commands

Command Arguments Description
pack <source> <archive> Compresses files/directories into ZIP, 7z, or tar.gz format.
unpack <archive> [output_dir] Securely extracts archive formats.
doctor [archive] Performs integrity checks and healing on self-healing archives.

Sandboxing Details

Decompression engines are notoriously complex and present a large surface area for memory safety vulnerabilities. Morphball deploys defense-in-depth measures:

Landlock LSM Sandboxing

During extraction, Morphball enforces Landlock sandboxing on the worker thread. It dynamically restricts process capabilities so that:

  • Write access is strictly bound to the target output directory.
  • Read access is limited to the input archive and essential system dynamic libraries.
  • All other directory write and network operations are blocked.
